# Лабораторна робота №1

## Тема
Розробка елементарних програм на мові програмування C#.

## Опис
Консольна програма вводить два числа, виконує математичні обчислення та демонструє:
- введення і виведення даних;
- конвертацію типів;
- використання класу `Math`;
- зміну кольорів у консолі;
- обробку помилок введення.

## Запуск
```bash
dotnet run
```
